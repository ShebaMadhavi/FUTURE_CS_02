# FUTURE_CS_02
# 🚀 Mini CRM - Client Lead Management System

## 📌 Project Overview
Mini CRM is a simple **Client Lead Management System** developed as part of the **Full Stack Web Development Internship (Future Interns - Task 2)**.

This project helps users manage, track, and update client leads efficiently through a clean and modern dashboard.

---

## 🌐 Live Demo
🔗 [https://your-netlify-link.netlify.app](https://idyllic-tiramisu-c64af6.netlify.app/)

---

## 🎯 Features
- 🔐 Login Authentication (Demo-based)
- 📊 Dashboard with lead statistics
- ➕ Add new leads (Name, Email, Source)
- 🔍 Search leads
- 🔄 Update lead status (New / Contacted / Converted)
- ❌ Delete leads
- 💾 Data stored using LocalStorage

---

## 🛠️ Tech Stack
- HTML
- CSS
- JavaScript
- LocalStorage (for data persistence)
